This project is an interactive **console-based quiz game** built entirely using JavaScript.  
It runs inside the **browser console**, asks the user multiple questions, processes input, gives immediate feedback, and displays the final score.

This project demonstrates:
- Arrays  
- Objects  
- Loops  
- Functions  
- Conditionals  
- String normalization using `toLowerCase()` and `trim()`  

No HTML or DOM manipulation is required.

---

## â­ Features
- Stores questions and answers using an **array of objects**  
- Takes user input using **prompt()**  
- Displays feedback using **alert()**  
- Normalizes user input with `toLowerCase()` and `trim()`  
- Shows instant correct/incorrect feedback  
- Tracks **score + percentage**  
- Displays performance messages  
- Allows the user to **restart the quiz**  
- Fully console-based (no HTML needed)

## ğŸš€ How to Run
1. Open any modern browser (**Chrome**, **Firefox**, **Edge**, etc.)  
2. Open the **Developer Console**:  
   - Windows/Linux: `Ctrl + Shift + J`  
   - Mac: `Cmd + Option + J`  
3. Open the file `quiz.js`  
4. Copy all contents of `quiz.js`  
5. Paste the code into the console  
6. Press **Enter** to start the quiz  
7. Answer the questions when prompted  
8. View your:  
   - Score  
   - Percentage  
   - Performance feedback  
9. Choose **Yes/No** if you want to restart the quiz

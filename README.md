# web-dev-4-quiz-game
This is an interactive console-based quiz game built using JavaScript. It runs entirely in the browser console, asks multiple questions, gives immediate feedback, and tracks the player's score. This project demonstrates core JavaScript concepts such as arrays, objects, loops, functions, string manipulation, and conditionals.
# Features
1)uses an array of objects to store questions and answers.
2)Prompts the user for input using prompt() and provides feedback using alert().
3)Automatically normalizes user input with toLowerCase() and trim() for accurate comparisons.
4)Provides instant feedback for correct and incorrect answers.
5)Tracks score and percentage at the end of the quiz.
6)Displays performance messages based on results.
7)Includes a restart option to retry the quiz.
8)Simple, clean, and fully console-based—no HTML or DOM required.
# How to Run
1)Open a modern browser (Chrome, Firefox, Edge, etc.).
2)Open the console:
   a) Windows/Linux: Ctrl + Shift + J
  b) Mac: Cmd + Option + J
3)Copy the content of quiz.js and paste it into the console.
4)Press Enter to start the quiz.
5)Answer each question when prompted.
6)View your score, percentage, and performance message at the end.
7)Choose whether to restart the quiz when prompted.
